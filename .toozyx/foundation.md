Business value.

==========================================================

Always build production-ready code.

Never build temporary solutions.

Never create technical debt intentionally.

Every implementation should be scalable.

Every implementation should be maintainable.

==========================================================

==========================================================

Readable.

Simple.

Reusable.

Modular.

Consistent.

Predictable.

Avoid clever code.

Prefer obvious code.

==========================================================
COMPONENT RULES
==========================================================

Every component should have one responsibility.

Avoid giant components.

Reuse components whenever possible.

Never duplicate logic.

If code is repeated three times,
create a reusable component.

==========================================================
FILE STRUCTURE
==========================================================

Keep folders organized.

Group related files together.

Use meaningful names.

Avoid generic names such as:

temp

new

test2

copy

final-final

==========================================================
PERFORMANCE FIRST
==========================================================

Performance is a feature.

Optimize before adding animations.

Optimize before adding effects.

Optimize before adding libraries.

Every page should load as fast as possible.

==========================================================
IMAGE RULES
==========================================================

Images must be optimized.

Use modern formats whenever possible.

Lazy load non-critical images.

Compress assets.

Avoid oversized images.

==========================================================
VIDEO RULES
==========================================================

Videos should never block rendering.

Lazy load embedded videos.

Load YouTube only when needed.

Optimize thumbnails.

Never autoplay large videos.

==========================================================
ANIMATION RULES
==========================================================

Animations should never reduce performance.

GPU-friendly animations only.

Prefer opacity and transform.

Avoid layout shifts.

RESPONSIVE DEVELOPMENT
==========================================================

Desktop.

Laptop.

Tablet.

Mobile.

All devices are first-class citizens.

Never fix desktop first

Semantic HTML.

Proper headings.

Labels.

Alt text.

Keyboard navigation.

Readable contrast.

Focus states.

Accessibility is mandatory.
